## Edge function modes

`generate-letter` accepts `mode=`:
- `individual&token=<uuid>` — per-invitee letter
- `joint&campaign=<slug>` — consolidated letter with all "supported" signatories
- `statewide&campaign=<slug>` — single-author letter from "the CPL Initiative at the Chancellor's Office"
- `preview&campaign=<slug>` — unfilled template + statewide stats only

`letter-curator` accepts:
- `GET ?campaign=<slug>` — all blocks for the campaign
- `GET ?campaign=<slug>&block_key=<k>&history` — version history
- `GET ?list=campaigns` — every campaign (drives the curator dropdown)
- `PUT { campaign_slug, block_key, scope, content }` — upsert
- `DELETE { campaign_slug, block_key, scope }` — delete (revert to scope=all)
- `POST { name, slug, letter_type, subject_line, deadline?, ask_amount?, blocks }` — create campaign + bulk-seed

All writes require `Authorization: Bearer <CURATOR_PASSCODE>`.
